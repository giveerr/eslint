## Install

You can install `@giveerr/prettier-config` via your package manager of choice.

### npm

```bash
npm install --save-dev @giveerr/prettier-config
```

### yarn

```shell
yarn add --dev @giveerr/prettier-config
```

### pnpm

```shell
pnpm add --save-dev @giveerr/prettier-config
```
